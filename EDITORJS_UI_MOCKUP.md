## EditorJS Editor Features

1. **Click to Add Block**: Click the "+" button or press Tab
2. **Drag & Drop**: Reorder blocks by dragging
3. **Inline Formatting**: Select text to format (bold, italic, marker, code)
4. **Block Conversion**: Convert blocks to different types
5. **Keyboard Shortcuts**: 
   - Tab: Create new block
   - Enter: Create new block (context-dependent)
   - Backspace: Delete empty block
   - CMD+SHIFT+M: Marker
   - CMD+SHIFT+C: Inline code

## Visual Styling

EditorJS maintains a clean, minimal aesthetic:
- Border: 1px solid neutral stroke
- Border radius: 4px
- Padding: 12px
- Font: System body font
- Background: White/transparent
- Hover states: Subtle highlighting
- Active states: Blue accent color
